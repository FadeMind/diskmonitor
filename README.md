# DisKMonitor
KDE tools to monitor SMART devices and MDRaid health status. Features a full application and a Plasma applet.

# Requirements
* **KF5**
* **UDisks2** >= 2.1

# Build
    mkdir build && cd build
    cmake ..
    make
